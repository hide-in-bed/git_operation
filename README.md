git_operation
$ git add * //把本地仓库的文件上传到缓存。
$ git commit -m 'del' //把第一步上传到缓存的东西上传到本地仓库，其中'del'是操作标识，内容随便填，方便用户观看。
$ git push origin master //把本地仓库的文件上传到远程仓库。
